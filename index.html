นายนิพัฒน์ แก้วมาตย์
การได้ร่วมงานกับ Incident Team เป็นสิ่งที่ภาคภูมิใจมากที่สุด ในความเป็น One Team ของเราที่แข็งแกร่ง และพวกเราร่วมกันสร้างความเป็น Number One อยู่เสมอมา ไม่เพียงเท่านี้ ในยามวิกฤต เราก็จะมีทั้งผู้บริหาร หัวหน้างาน คอยฉายแสงนำทาง และเพื่อนร่วมงาน ที่ร่วมทุกข์และสุขด้วยกันอยู่เสมอ
นายชาตรี นทีพงศ์พริยะ
ทุกช่วงเวลาที่นี่ล้วนแสดงออกถึงความเป็น One Team แต่ถ้าเป็นเหตุการณ์ที่ติดตรึงใจมากที่สุดน่าจะเป็นเหตุการณ์ไฟฟ้าดับที่ตึก IT DATA CENTER อาคารราษฎร์บูรณะ เมื่อปี 2015 ตอนนั้นเราทุกคนต้องร่วมมือร่วมใจกัน Recover ให้สถานการณ์กลับมาเป็นปกติได้อย่างอย่างรวดเร็วที่สุด บางคนโดนโทรปลุกกลางดึก เราต้องรีบเข้ามา Office เพื่อร่วมมือกัน บางคนต้องขึ้นไปชั้น 3 เพื่อ Restart server บางคนต้องอยู่ชั้น 2 เพื่อ Restart system ต่างๆ ให้ขึ้นมา เหตุการณ์นี้เป็นเหตุการณ์ที่ผมและเพื่อนๆ น้องๆ จดจำได้ตลอดไม่ลืม ว่าผ่านพ้นมาได้ด้วยความเป็น One Team ของพวกเราจริงๆ
นางนงลักษณ์ พิทักษ์พาณิชกุล
การทำงานที่นี่ เราไม่ได้มีแค่เรื่องงานเพียงอย่างเดียว แต่ที่นี่ยังมีเพื่อน พี่ น้อง และมีกิจกรรมดีๆให้พวกเราได้มีโอกาสตอบแทนสังคมของเราได้อีกด้วย พี่ประทับใจกับกิจกรรมหลากหลายอย่างที่ทำให้พี่ได้มีส่วนร่วมในการช่วยเหลือสังคม ในแต่ละครั้งที่เราร่วมกิจกรรมกัน เรามี One Goal ที่เป็นหนึ่งเดียว ที่อยากร่วมแรงร่วมใจทำเพื่อสังคมจริงๆ ไม่ว่าจะเป็น การเย็บเต้านมเทียมสำหรับผู้ป่วยมะเร็ง การจัดทำดอกไม้จันทน์ถวายพ่อหลวง การร่วมกิจกรรมทำดีกับเอ็มดีปู ที่บ้านราชาวดีหญิง เป็นประสบการณ์ที่ดี และทุกกิจกรรมนั้น เรารวมตัวเป็น One Team ได้อย่างดีเยี่ยม ทุกคนรักใคร่ กลมเกลียวกัน และล้วนมีความสุขทุกครั้งที่ได้มีส่วนหนึ่งในกิจกรรมดีๆ เพื่อสังคม และเพื่อองค์กรของเราค่ะ
นายสุชาติ ทับทิมศรี
ความเป็น ONE KBTG ของพวกเราชัดเจนมาก และผมเชื่อว่าทุกๆคนใน KBTG ก็เป็นเช่นนั้น ทั้ง One Step Ahead, One Goal, One Team และ Number One ทุกอย่างนี้มันเป็นพื้นฐานให้พวกเราทำ และต่อยอดถึงผลสำเร็จได้อีกมากมาย สิ่งที่ชัดเจนสำหรับผมคือผมได้ร่วมงานกับทีมงาน ATM ผมได้รับการดูแลจากหัวหน้างานและทีมงานเป็นอย่างดี ทุกคนสามัคคี และทุ่มเทการทำงานอย่างสุดความสามารถ จนในปี 2019 สิ่งที่พวกเราทำนั้น ได้ประสบความสำเร็จขึ้นไปอีกขั้น โดยพวกเราได้ขึ้นรับรางวัลประจำปีกันอย่างต่อเนื่องในหลายๆรางวัล ผลการทำงานของทีมเราได้พิสูจน์ความเป็น One KBTG เราสามารถทำให้ Service AVA ดีขึ้นและปัญหาต่าง ๆ ลดลงอย่างต่อเนื่อง ก่อให้เกิดความพึงพอใจแก่ลูกค้าของธนาคารที่เข้ามาใช้บริการ
นายวิโรจน์ รุ่งเพชรรัตน์
ที่นี่มอบโอกาสในการทำงานที่หลากหลายให้กับทุกคน ครั้งหนึ่งผมได้รับหน้าที่เป็น Project Owner ในการทำกิจกรรม KBTG eSport ROV ซึ่งเป็นสิ่งที่ท้าทายสำหรับตัวผมมากในการจัดกิจกรรมให้พนักงานมารวมตัวกันทำงาน วางแผน คิดวิเคราะห์ เพื่อแสดงถึงความเป็น One KBTG และ Project นี้เอง ทำให้ผมได้มีโอกาสสร้างสิ่งดีๆ รอยยิ้ม และความประทับใจให้กับพนักงานทุกคนที่ร่วมกิจกรรม ทุกคนได้ร่วมกันสร้างความเป็น One Goal, One Team และ Number one กันอย่างเต็มที่ และสนุกสนานผ่านกิจกรรมที่ผมจัดนี้ นับเป็นความทรงจำ และความประทับใจที่ผมไม่มีวันลืมครับ
นายวีระศักดิ์ ฐิติเบญจพล
ผู้บริหารทุกคนที่นี่เอาใจใส่พนักงานทุกคนเป็นอย่างดี นโยบายหลายๆอย่างที่ผู้บริหารทำให้พนักงาน พวกเรารับรู้ และสัมผัสได้ในความผลักดันให้เกิด One KBTG สำหรับพวกเรา และผมมั่นใจว่า พวกเราทุกคนจะไม่ทำให้ผู้บริหารผิดหวังในตัวพนักงานอย่างพวกเราแน่นอนครับ
นายธนฤกษ์ เกาะศิริ
One KBTG ทำให้ผมรู้สึกว่า พวกเราเป็นครอบครัวเดียวกันจริงๆครับ ที่นี่เป็นครอบครัวที่อบอุ่น เป็นองค์กรที่มั่นคง พนักงานสามารถฝากอนาคตไว้กับ KBTG ที่เป็นครอบครัวใหญ่ร่วมกันกับ "ครอบครัวกสิกรไทย" ตัวผมสังกัด KServe ภรรยาของผมสังกัด KSoft ทั้งผมและภรรยา เรารักและประทับใจในความดูแลเอาใจใส่ครอบครัวของผมเป็นอย่างดีตลอดมาครับ
นายเอนก คนซื่อ
ที่ KBTG นโยบาย Number One ทางด้าน IT และ นวัตกรรมชัดเจนมากครับ ผมในฐานะพนักงานคนหนึ่ง ผมดีใจที่เป็นส่วนหนึ่งในการนำเครื่องมือ และนวัตกรรมใหม่ๆมาใช้ในการทำงาน และผมรู้สึกได้อย่างชัดเจนว่า เรา One Step Ahead อยู่เสมอ ไม่ว่าในสถานการณ์ไหน หรือวิกฤตการณ์ใด ผมตัดสินใจไม่ผิดที่มาร่วมงานกับ KBTG ครับ
นายพีระพงศ์ พูลผล
การร่วมงานกับ KBTG ทำให้ผมได้มีโอกาสทำอะไรหลายๆอย่าง ที่ไม่ใช่แค่เรื่องงานอย่างเดียว ผมมีโอกาสได้รู้จักเพื่อนใหม่ๆ มีโอกาสได้ทำอะไรใหม่ๆ ทั้งเรื่องงาน และกิจกรรมที่องค์กรมอบให้พนักงาน ที่นี่เป็นที่เรียนรู้ที่ดีสำหรับคำว่า One Team เราเป็นทีมที่แข็งแกร่งมากครับ ทั้งพี่ๆ เพื่อนๆ น้องๆ ทุกคน ล้วนสนุกไปด้วยกันกับสิ่งที่เราทำ ด้วยความรัก และความสามัคคีครับ
